Foota Mashinani API

Product concept

Make our Grassroots Football API, your go-to source for comprehensive data on grassroots football in Kenya! Dive into the heart of the game by accessing a wealth of information, including club details, player profiles, league standings, and match fixtures. Our API empowers you with JSON-formatted data, meticulously gathered from various sources, providing a reliable and up-to-date overview of the vibrant grassroots football landscape in Kenya. Whether you're a developer, football enthusiast, or league administrator, our API opens the door to a rich tapestry of football data, facilitating seamless integration and exploration of the dynamic world of grassroots football in Kenya.

How to make an API call

In Django

               
                  # Python


                  import requests

                  url = 'https://foota.pythonanywhere.com/api/v1/the-endpoint/'
                  api_key = 'your_api_key'
                  
                  headers = {
                     'X-API-KEY': api_key,
                  }
                  
                  response = requests.get(url, headers=headers)
                  
                  if response.status_code == 200:
                     data = response.json()
                     print(data)
                  else:
                     print(f"Error: {response.status_code}")
